# How to Embed SmartLead Magnet into Your HubSpot Website Pages

**Note:** To follow this guide, you should have administrative access to your HubSpot account.

## 1. Log in to Your HubSpot Account

Go to your HubSpot login page: `https://app.hubspot.com/login`.

Enter your username and password to access your dashboard.

## 2. Navigate to Website Pages

In your HubSpot dashboard, go to **Content** > **Website** > **Website Pages**.

## 3. Create or Edit a Page

- Click **Create** to start a new page, or click **Edit** on an existing page where you want to embed the iFrame.

## 4. Add an Embed Code Module

In the page editor, look for the **Rich Text** module.

## 5. Paste the iFrame Code

Copy the iFrame code from the SmartLead Magnet platform.

Paste the iFrame code into the Embed Code module you just added.

Your iFrame code should look something like this:

```html
<iframe src="https://smartleadmagnet.com/share/magentid" width="100%" 
    height="600" 
    style="border-radius:0.5rem; border:none; 
    max-width: 100%; 
    height: 70vh; /* Responsive height based on viewport height */
    min-height: 500px;" 
    loading="lazy"
    referrerpolicy="unsafe-url" 
    allow="clipboard-read;clipboard-write"
></iframe>
